# Images

Add optimized photos here (WebP or JPEG). Keep file sizes small for fast loads on mobile.

Suggested files (update paths in `index.html` if you rename):

| File | Purpose |
|------|---------|
| `hero.jpg` or `hero.webp` | Main hero portrait / press photo (roughly 1200px wide is enough). |
| `LogoMK.jpg` | Master logo (white background). Header uses `logo-header.jpg` (128×128, from this file). |
| `logo-header.jpg` | Nav mark — regenerate with `sips` if you replace `LogoMK.jpg`. |
| `LogoMK.png` | Optional transparent master (not used for header when using JPG). |
| `favicon-16.png`, `favicon-32.png`, `apple-touch-icon.png` | Tab / home-screen icons (generated from `LogoMK.jpg`). |
| `logo.svg` | Legacy simple mark (unused if you use `logo-header.jpg`). |
| `gallery-01.jpg` … | Square or landscape shots for the photo strip (delete unused figures in HTML). |

**Tips:** Export at 72–85% JPEG quality or use WebP. Run images through a compressor (e.g. Squoosh) before committing.
